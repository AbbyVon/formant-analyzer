Formant Plotter

This application plots the formant.

frequencies of spoken vowel sounds.

The user should speak a word containing a vowel, pause while application process the sound, and see results.

First two formant frequencies are used to graphically show the location of spoken vowel on a standared vowel diagram.

An indicator light changes its colour to display the status of the application.
