Formant Plotter

This application calculates and plots the formant frequencies of spoken vowels. The user should speak a word containing a vowel, pause while application processes the sound, and see results.

First two formant frequencies are used to graphically show the location of spoken vowel on a standared vowel diagram. An indicator light changes its colour to display the status of the application.
